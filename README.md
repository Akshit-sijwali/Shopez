🛍️ Shopez
Simple Affiliate E-Commerce Website

Shopez is a lightweight affiliate e-commerce website that showcases products and redirects users to partner platforms for purchase.

🚀 Features
🛒 Browse curated products
🔗 Affiliate links for external purchase
⚡ Fast and minimal UI
📱 Responsive design
🧱 Tech Stack
Frontend: HTML, CSS, JavaScript
---
Hosting: GitHub Pages / Vercel
📂 Project Structure
shopez/
│── index.html
│── style.css
│── script.js
│── assets/
⚙️ Setup & Run
Clone the repository
Open the project:
cd shopez
Run locally:
Open index.html in your browser
💰 How It Works
Users browse products on Shopez
Clicking a product redirects to affiliate platforms
You earn commission on successful purchases
